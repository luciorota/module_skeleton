Help: Module_skeleton

Fresh Install

These instructions are for a fresh install.

If you want to update an older Module_skeleton please select 'Update'.
If you want to import from Xoops myDownloads please select 'Import'.

We recommends you take a full backup before INSTALLING or UPDATING any module.

Fresh installation of Module_skeleton

Upload the module to your website

Upload the 'module_skeleton' folder to {xoops-rootdirectory}/modules folder.

Change and verify folder permissions

CHMOD the following folders to 777:
module_skeleton/assets/images
module_skeleton/assets/images/itemcategory
module_skeleton/assets/images/itemcategory/thumbs
module_skeleton/assets/images/screenshots
module_skeleton/assets/images/screenshots/thumbs

Also make sure that the {xoops-rootdirectory}/uploads folder is set to 777.

Install the module

Login as administrator and enter Xoops Administration page.
Select System --> modules and install module_skeleton from there.

On Linux/Apache servers, in order to avoid any kind of problem in audio/video preview, create a filed called .htaccess with the following text and upload it to the root of your website.   AddType video/ogg .ogv   AddType video/mp4 .mp4   AddType video/webm .webm

Configure the module

Most important steps now will be to setup the group permissions for the module and its blocks via System --> groups and than within module_skeleton administration Mimetype management to decide which files are allowed to be uploaded by users and administrators.
